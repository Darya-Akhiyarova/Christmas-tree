# Christmas-tree

**Создание кастомного брелока со встроенными светодиодными лампочками**
--------------------------
*Оборудование: 
3D-принтер Prusa i3 MK3
Лазерный гравер GCC Spirit GLS100
Сверлильный станок Bosch PBD 40
Паяльник*

*Материалы и элементы:
Фанерный лист
PLA (полилактид)
Светодиоды
Медный провод
Припой (олово)
Батарейка CR2032 (2V)*

*Программное обеспечение:
PrusaSlicer
CorelDraw*
